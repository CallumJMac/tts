Multi-Reference Voice Cloning for Qwen3-TTS

Audio demos · A Trade-Off Between Speaker Fidelity and Naturalness

When multiple reference recordings are available, how should they be combined? 1,275 controlled synthesis runs on Qwen3-TTS reveal a clear Pareto trade-off:

Maximise speaker identity
ConcatAudio, longest 3 refs
SIM +0.02 · d = 0.62 · p < 0.001
Maximise naturalness
EmbedAvg, any 3 refs
UTMOS +0.06 · d = 0.61 · p < 10⁻⁷
