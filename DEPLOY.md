# Listening Study Deployment Guide

## Prerequisites

- Python 3.9+ with pandas, numpy
- sox (for anchor generation): `brew install sox`
- Audio files from phase4_40spk experiment accessible locally

## Setup Steps

### 1. Generate stimuli and config

```bash
cd listening_study/
python select_stimuli.py \
    --results ../results/phase4_40spk/results.csv \
    --results-base ../results/phase4_40spk \
    --output-dir . \
    --n-trials 12
```

This will:
- Select 12 high-contrast trials + 1 practice trial (balanced across speakers)
- Copy WAV files into `audio/`
- Generate time-stretched anchors (0.7x speed via sox)
- Create headphone check tone
- Build the final `configs/study.json`

### 2. Set up results collection (Google Sheets)

1. Create a Google Sheet
2. Go to Extensions > Apps Script
3. Paste the code from `google_apps_script.js`
4. Deploy as Web App (execute as: Me, access: Anyone)
5. Copy the deployment URL into `configs/study.json` > `resultsEndpoint`

### 3. Test locally

```bash
python -m http.server 8000
# Open http://localhost:8000
```

### 4. Deploy to GitHub Pages

```bash
# From repo root
git subtree push --prefix listening_study origin gh-pages
# Or copy listening_study/ contents to a gh-pages branch
```

The study is entirely static (HTML/CSS/JS + audio files). Any static host works.

### 5. Prolific setup

- Study URL: `https://<your-pages-url>/index.html?PROLIFIC_PID={{%PROLIFIC_PID%}}`
- Completion code: `VCMUSHRA2026`
- Estimated time: 10 minutes
- Reward: ~GBP 2.00 (GBP 12/hr)
- Prescreening: Fluent English, normal hearing, desktop device

## Design Decisions

- **12 trials**: ~10 min at 45s/trial. Balances statistical power vs. fatigue.
- **High-contrast selection**: Stimuli chosen where concat_audio and embed_avg diverge most on automated metrics, maximising chance of perceptual differences.
- **Two scales per trial**: Naturalness and Speaker Match rated independently (validates the fidelity-naturalness trade-off from automated metrics).
- **Anchor**: 0.7x time-stretch of reference. Clearly degraded but recognisable.
- **Condition randomisation**: Order shuffled per trial per participant.
